Keyboard shortcuts are used by typing the key while the mouse cursor is in the appropriate view. You do not have to select the view before typing the key.

Visual Filter Shortcuts

  • 1 or End: zoom in in Y (and X, if a scatter plot)
  • 2 or down arrow: move the visual filter down
  • 3 or Page Down: zoom out in Y (and X, if a scatter plot)
  • 4 or left arrow: move the visual filter left
  • 5: set the visual filter to the "home" location (the default is to show all data)
  • 6 or right arrow: move the visual filter right
  • 7 or Home: zoom in in X (and Y, if a scatter plot)
  • 8 or up arrow: move the visual filter up
  • 9 or Page Up: zoom out in X (and Y, if a scatter plot)
  • Backspace: toggle between the last two visual filter settings

Symbol Display Shortcuts

  • +: increase symbol size
  • -: decrease symbol size
  • v or V: toggle display of data values as text superimposed on symbols
  • c or C: toggle use of connectors (disabled)
  • s or S: toggle display of symbols (disabled)
