Help about this visualization

Switching visualizations: to switch between visualizations for this entry, select Session->Open/Switch in the JavaScreen. This will show a dialog listing the visualizations that are currently available for this entry. Note that some visualizations (such as coordinates and restraints) require additional processing on the server, so you may have to go back to the "all visualizations" page for this entry to access them. Also, if you switch visualizations inside the JavaScreen, the "help on this visualization" link will no longer correspond to the visualization you are viewing.
