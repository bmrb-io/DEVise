BMRB two-entry visualizations



You can also upload data to the Visualization Server and compare it to a released entry (first upload your data to the visualization server and generate the visualizations; once you've done that you can enter an accession number for generating the two-entry visualizations).


Currently two two-entry visualizations are available: the simulated 1H-15N backbone HSQC spectrum and the simulated 1H-13C HSQC spectrum. Examples of each are shown below, with links to the help pages offering detailed explanations.
