Chemical Shift Delta

The plotted chemical shift differences have been calculated using the observed chemical shifts reported in the entry and the average chemical shift for the atom derived from the BMRB database (see http://www.bmrb.wisc.edu/ref_info/statsel_dna.htm and http://www.bmrb.wisc.edu/ref_info/statsel_rna.htm).

Main Visualization Window

The visualization is laid out as shown below; the view names are referred to in the text.

Data view 1 shows chemical shift deltas for one combination of nucleotide and atom.

Nucleotide select view 1 selects the nucleotide (or all nucleotides) for which data is shown in data view 1.

Atom select view 1 selects the atom for which data is shown in data view 1. The associated scroll view scrolls the atom select view if there are too many atoms to fit within the view.

Data view 2, nucleotide select view 2, and atom select view 2 function similarly to the first set of views, allowing comparison of data for two different combinations of nucleotide/atom.

The info view shows the BMRB accession number and generation date of the visualization; show the view help (F1 key) to see the Peptide-CGI version and exact generation timestamp.
