Keyboard Shortcuts

Keyboard commands are used by typing the key while the mouse cursor is over the appropriate plot. It is not necessary to click in the plot before typing the key. The data record corresponding to a data point can be obtained by holding the Shift key while clicking on that data point in a plot.

  • Reset to original view: 5
  • X zoom in: 7 / Home
  • X zoom out: 9 / Page Up
  • Y zoom in: 1 / End
  • Y zoom out: 3 / Page Down
  • Move visible area up: 8 / up arrow
  • Move visible area down: 2 / down arrow
  • Move visible area left: 4 / left arrow
  • Move visible area right: 6 / right arrow
  • Toggle between last two settings: Backspace
  • Increase symbol size: +
  • Decrease symbol size: -
  • Toggle display of data text: v
