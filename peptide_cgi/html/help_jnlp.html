Java Web Start

At the end of 2016 major browser vendors disabled NPAPI support which was required to launch the DEVise visualizations directly from the browser. The DEVise visualization tool is still available but it now requires a few extra steps for you access the visualizations.

macOS Sierra

Regardless of which browser you use, you will need to navigate to the location of the downloaded file in the finder and then open it from there. New macOS security rules prevent you from launching the file directly after download.

Here is an example of how to launch DEVise from Safari:

  1. Navigate to the visualization you wish to view, and click the "Click here to launch DEVise for this visualization" text:
  2. A file ending in .jnlp will download. You cannot click on it directly from the downloads list - macOS will not allow you to open it that way.
  3. Open up the Finder and navigate to the location of the file you downloaded.
  4. Either right click or control-click on the .jnlp file and select "Open With" -> "Java Web Start":

  5. Click "Open" on the dialog box that appears.

If you are still unable to launch the application after following these steps, go to System settings -> Java -> Security and change the setting from "Very High" to "High".
