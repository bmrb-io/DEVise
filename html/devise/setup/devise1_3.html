Architectures

DEVise 1.3.4 is available for the following architectures:

  • Intel/Solaris
  • SPARC/Solaris
  • SPARC/SunOS
  • HPPA/HPUX

Setup Procedure

To set yourself up to run on any of these platforms, first add /unsup/devise-1.3/bin to your path (in your .cshrc.local file if you're running c shell, for example). Then run 'devise.setup'. This script creates a devise directory within your home directory, and copies a number of files (such as sample data files) into that directory.

Once you've run devise.setup once, you can run DEVise simply by entering the command 'devise'.
